FVReaction

Description

FVReaction implements a simple first-order reaction term with unity rate coefficient where the rate of reaction is directly proportional to the governing variable .

FVReaction can be used to help set-up variations of advection-diffusion-reaction equations.

Example Syntax

The syntax for FVReaction is simple, only taking the type and variable parameters. An example block is shown below for a diffusion-reaction equation:

[FVKernels]
  [diff]
    type = FVDiffusion
    variable = v
    coeff = coeff
  []
  [rxn]
    type = FVReaction
    variable = v
  []
  [diffw]
    type = FVDiffusion
    variable = w
    coeff = coeff
  []
  [prod]
    type = FVCoupledForce
    variable = w
    v = 'v'
  []
[]
