VectorCoupledTimeDerivative

Time derivative Kernel that acts on a coupled vector variable. Weak form: .

Overview

The VectorCoupledTimeDerivative kernel is very similar to the VectorTimeDerivative kernel with the exception that the time derivative operator is applied to a coupled variable instead of the variable to whom's residual the VectorCoupledTimeDerivative kernel contributes. Example Input File Syntax

[Kernels]
  [./time_u]
    type = VectorTimeDerivative
    variable = u
  [../]
  [./fn_u]
    type = VectorBodyForce
    variable = u
    function_x = 1
    function_y = 1
  [../]
  [./time_v]
    type = VectorCoupledTimeDerivative
    variable = v
    v = u
  [../]
  [./diff_v]
    type = VectorDiffusion
    variable = v
  [../]
[]

    boundary = 'right'
    values = '1 1 0'
  [../]

  solve_type = 'NEWTON'
[]
