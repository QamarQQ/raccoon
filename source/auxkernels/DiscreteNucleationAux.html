DiscreteNucleationAux

Project the DiscreteNucleationMap state onto an AuxVariable

This AuxKernel computes the value as

where ist the variable the auxkernel is acting on and is the DiscreteNucleationMap data. The map values range from 0..1 and are remapped using the (no_nucleus_value) and (nucleus_value) parameters.

The DiscreteNucleationAux auxkernel is part of the Discrete Nucleation system. Its companion kernel is DiscreteNucleationForce.
