MortarPeriodicity System

The MortarPeriodicity Action assists users in setting up mortar based periodicity constraints. Its primary application is setting up an EqualGradientConstraint which, when used on the displacement variables, implemnts a strain periodicity which is useful for representative volume element microstructure simulations.

Available Actions

  • Phase Field App
  • MortarPeriodicActionAdd mortar interfaces, Lagrange multiplier variables, and constraints to implement mortar based periodicity of values or gradients on a MortarPeriodicMesh
