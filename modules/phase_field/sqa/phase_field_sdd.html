Phase Field Software Design Description

Introduction

The SDD for Phase Field is a description of the software product and serves as guideline that describes the architecture of the system and all of its parts.

Dependencies

The Phase Field application is developed using MOOSE and is based on various modules, as such the SDD for Phase Field is dependent upon the following documents.

  • Framework System Design Description
